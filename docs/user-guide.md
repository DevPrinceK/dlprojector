# DL Projector User Guide

## Starting Service

1. Open DL Projector.
2. Click `Open Projection`.
3. Move the projection window to the projector display.
4. Click `Fullscreen`.
5. Use the control window to search, preview, and project content.

## Projection Safety

- `Blank` immediately shows a black screen.
- `Logo` returns to the DLCF Legon logo.
- `Emergency Reset` clears unsafe operator state and sends a stable logo slide.
- The projection window keeps the last valid slide if a new slide fails to render.

## Common Workflows

- Scriptures: search references such as `John 3:16`, `Psalm 23`, or `Genesis 1:1-5`.
- Hymns: add hymns manually, split stanzas with blank lines, and use `Chorus:` for chorus text.
- Announcements: create a title, concise message, date, time, venue, and optional image path.
- Personality: create a weekly member profile with photo, role, department, bio, and favorite scripture.
- Service Program: build the order of service and project each item from the list.
